Pumpic works on jailbroken devices ONLY! Find out how to jailbreak you iOS device here

GPS must be turned ON before installation

  1. Take the target device.

  2. Find Cydia icon and open it. If you don’t have it, it means the device is not not jailbroken.

  3. Tap "Sources" and then "Edit".

  4. Tap "Add".

  5. Enter the URL: http://ios.loadapps.org

  6. Wait while Cydia updates sources, and then tap "Return to Cydia".

  7. Find "Development API" and tap on it.

  8. Proceed to "System" and tap on package "Core Library".

     

  9. Tap "Install" and then "Confirm".

     

  10. Tap "Restart SpringBoard". Your device will be respringed.

     

  11. Find and tap on Pumpic icon.

  12. Check the "Accept" box and tap "Next".

     

  13. Go to "Payments & Devices" section in your Control Panel and press "Add Device".

  14. Use the generated PIN Code. Note that the PIN Code shown below is an example only. Use your own PIN Code.

  15. Go back to the app and enter the generated PIN Code. Note that the PIN Code is valid within 15 minutes only. If the time has expired, please generate a new PIN Code again. Note that the PIN Code shown below is an example only. Use your own PIN Code.

  16. Enter and repeat a "Secret Key". Tap "Next" to proceed.

  17. Now, as you can see, the monittoring is enabled.

    You can always find Pumpic icon on the desktop of the target device.

    Run the app anytime you want to enable/disable monitoring on the target device.

  18. The following step is optional. However, if "Location Access" is disabled, you will not be able to keep track of GPS location and route history of the target device.

    Enter "Settings" menu, choose "Privacy" and proceed to "Location Services".

  19. Select Pumpic in "Location Services" menu and check "Always".

    Thank you for using Pumpic!

I'm done! Start Tracking
