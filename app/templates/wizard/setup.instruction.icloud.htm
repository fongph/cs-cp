{*

You can easily track the target iOS device without jailbreak. However, previously make sure that:

  1. iCloud Backup is enabled on the target iPhone, iPad or iPod Touch.

    To make sure, take the target Apple iOS device. Then go to Settings > Apple ID > iCloud > iCloud Backup. If iCloud Backup is off, please turn it ON (as shown below).

    Note:

    * For iOS 9-10.3.1, the path is Settings > iCloud > Backup

    * For iOS 6-8 path is Settings > iCloud > Storage & Backup

      

  2. You have Apple ID of the target iOS device and know the password.

    If both of the aforementioned requirements are satisfied, you can proceed to the next step of these guidelines.

    Note: iCloud backs up the data from the target device automatically once per 24 hours. In addition, the following requirements should be also satisfied:

    1. the target device is connected to Wi-Fi;
    2. it is connected to the power source;
    3. the screen is locked.

    It all means that the data from the target devices might be sent to your Control Panel with a day delay. If you need the data updates to be uploaded as soon as possible, you can perform iCloud Backup manually. Go to Settings > Apple ID > iCloud > iCloud Backup (as shown above) and tap Back Up Now.

     

    Wait till backup process is finished. This may take several minutes.
