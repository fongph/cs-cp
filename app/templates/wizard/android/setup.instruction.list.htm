• Take the target device.

    Make sure that "Unknown Sources" option is checked. This option can be found at one of the following locations depending on your device model:

    Settings > Security OR Settings > Applications

    The screenshots below show the checked "Unknown Sources" option on different models of devices. The screen of your own device may be a little different, but the path will match with one of the locations mentioned above.

    Settings > Security

    On devices with old Android version (up to 4.0), you should go to Settings > Applications.

  • Direct download:

    To download the application, it is better to use a standard Internet browser. Open a browser on the targeted device and enter the following URL: http://android.loadapps.org You can also use the QR Code below:

    Using the target device with an old version of Android, you need to tap "Option" button > "Save" > "OK".

      

  • Start the installation.

    Go back to "Apps" and choose "Downloads" section.

  • Tap on the downloaded monitoring.apk to start the installation process.

    If the installation file is not opening for your device, use ES file manager to open downloaded file.

  • Choose "Next".

  • Scroll the screen down and tap "Install" to start.

  • When the installation is completed, tap "Open".

  • (required for devices with Android 5.0+)

    Move the switch to the right to enable "Monitoring".

  • (required for devices with Android 5.0+)

    Tap "OK" to allow access.

  • (required for devices with Android 5.0+)

    Now "Monitoring" is enabled. Tap "Back" to return to the Pumpic app.

  • Tap "Activate" to proceed.

  • Check the "Accept" box and tap "Next".
