You can restrict account changes by using Apple Restriction feature on the target device to prevent Apple ID change or back up disabling by target device user.

Go to Settings > General > Restrictions and enter Restriction Code or create a new one. If a code has already been created and you do not know it, then the only way is to erase the device and set it as a new one.

Scroll down to Allow Changes and tap on Accounts. Select Don't Allow Changes.
