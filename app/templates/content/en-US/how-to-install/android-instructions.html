1. Take the target device.

    Make sure that "Unknown Sources" option is checked. This option can be found at one of the following locations depending on your device model:

    Settings > Security OR Settings > Applications

    Following screenshots show the checked "Unknown Sources" option on different device models. The screen of your own device may be a little different but the path will match with one of the locations mentioned above.

    Settings > Security

    On devices with old Android version (up to 4.0), go to Settings > Applications.

  2. Direct download:

    To download the application, it is better to use a standard Internet browser.

    Open a browser on the targeted device and enter the following URL: http://android.loadapps.org

    You can also use the QR Code below:

    Using the target device with an old version of Android, you need to tap "Option" button > "Save" > "OK".

      

  3. Start the installation:

    Go back to "Apps" and choose "Downloads" section.

    Tap on the downloaded pumpic.apk to start the installation process.

    Choose "Next".

    Scroll the screen down and tap "Install" to start.

    When the installation is completed, tap "Open".

    Tap "Activate" to proceed.

  4. Check the "Accept" box and tap "Next".

     

    Now the app will ask you to enter a PIN code, which you can generate in your Control Panel.

  5. Go to "Payments & Devices" section in your Control Panel and press "Assign Device".

    Use the generated PIN Code. Note that the PIN Code shown below is an example only. Use your own PIN Code.

    Go back to the app and enter the generated PIN Code. Note that the PIN Code is valid within 15 minutes only. If the time has expired, please generate a new PIN Code again.

    Enter and repeat a ''Secret Key''. Tap "Next" to proceed.

    Now, as you can see, the monittoring is enabled.

    You can always find Pumpic icon in Apps menu.

    Run the app anytime you want to enable/disable monitoring on the target device.

    The installation was completed correctly. Pumpic is already monitoring the target device. The data from the target device will soon be sent to your Control Panel.

I'm done! Start Tracking
