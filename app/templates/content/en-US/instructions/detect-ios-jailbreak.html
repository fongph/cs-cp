Jailbreak is required to install Pumpic on the target device. That’s why you should check if the target device is jailbroken before the installation.

  1. Go to Home Screen and type “Cydia” in Spotlight Search. If the Cydia icon appears, it will mean that the target device is jailbroken.

  2. If the first method doesn’t work, try another one.

    The most popular apps, which hide Cydia, reveal it as follows:

    • Open the Safari browser;

    • Type in the URL “cydia://”;

    • Press “Go”.

  3. If the aforementioned methods don’t show Cydia, there is still a chance it is installed.

    The best and most exact check is to download TaiG (the software that performs jailbreak) on your PC. The program will show you the status of the target device, and if it is not jailbroken, you will be able to install jailbreak.
