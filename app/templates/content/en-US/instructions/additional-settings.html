Since Pumpic does not have control over the target device with iOS (iCloud) Monitoring solution, we recommend you to adjust additional settings to get more Parental Control options. Here are a few quick tips:

1. Improve Browser History Monitoring

To disable private browsing and restrict adult websites:

Go to Settings > General > Restrictions and enter Restriction Code or create a new one. If a code has been already created and you do not know it, then the only way is to erase the device and set it as a new one.

Scroll down to Allowed Content and tap on Websites. Select Limit Adult Content. You can also allow and disallow certain websites bellow.

2. Block Unwanted Applications

Since it is not possible to monitor all apps with Pumpic, consider deleting all unwanted apps on the device and block possibility to install them without your approval:

Go to Settings > General > Restrictions and enter Restriction Code. Scroll down to Installing Apps and disable it. Now you can delete any app on the device. To install them back you need to re-enable Installing Apps feature, which is not possible without Restrictions Code.

3. Block In-App Purchases

To avoid unnecessary in-app purchases you can simply block this option:

Go to Settings > General > Restrictions and put in Restriction Code. Scroll down to in-app purchases and disable it. You can find a screenshot above.
