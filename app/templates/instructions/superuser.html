The target device seems to be rooted, but you have not granted the root access to Pumpic yet. The app requires Superuser rights periodically. Tap “Grant” to provide, when you see the following pop-up window.

If a root access pop-up window has not appeared yet, or you have denied the root access previously, please follow the instructions below:

  1. Install SuperSU application.

  2. Enter launcher and choose SuperSU.

  3. You will see "Monitoring" app in the list with a red hash # on the right. Tap on “Monitoring” application.

  4. A pop-up with the access rights will appear. Tap on “Deny” in the “Access” line.

  5. Chose “Grant” from the list.

  6. Tap “Save”.

  7. A green hash # will appear near "Monitoring" app.

Now when the root access is granted, you can enjoy the entire range of Pumpic monitoring features.

Are you done? Please wait for 20 minutes until Pumpic starts collecting data.
